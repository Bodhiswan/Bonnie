# Bonnie Mills — Website Wireframe

A static HTML/CSS wireframe for Bonnie Mills — artist, maker, and mother.

## Pages
- `index.html` — Home
- `about.html` — About Me
- `gallery.html` — Gallery (displays prices for reference; not a shop)

## Style
- Modern cursive headings (Google Fonts: Dancing Script / Parisienne)
- Subtle tiled native Australian flower background (`assets/bg-natives.svg`): waratah, golden wattle, flannel flower, kangaroo paw, gum blossom, eucalyptus
- Low-fidelity wireframe structure with dashed sections and image placeholders

## Preview
Open `index.html` directly in a browser. No build step.
